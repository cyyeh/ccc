uart · clint · plic · block

Three labs to play with the device emulation behind cccs interrupt path.

Push bytes into the UART FIFO; watch PLIC source 10 assert.

Mirrors uart.pushRx. Bytes accumulate in the 256-byte FIFO; the empty→non-empty transition asserts PLIC source 10.

Set mtimecmp. Tick mtime forward. Watch mip.MTIP flip.

Mirrors clint.isMtipPending. MTIP is computed live as mtime >= mtimecmp.
