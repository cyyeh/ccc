console & editor

Cooked-vs-raw simulator. ANSI escape sandbox. Editor key cheatsheet.

Type into a simulated console; toggle modes; see the difference.

In cooked mode, ^C/^U/backspace are handled by the line discipline. In raw mode, every byte goes straight to the program.

Type ANSI escape sequences. Watch the cursor move.

A 12×40 grid acts as a fake terminal. Type a sequence (e.g., \\x1b[5;10H) and any text after; click "Apply".


      

        
SequenceEffect
