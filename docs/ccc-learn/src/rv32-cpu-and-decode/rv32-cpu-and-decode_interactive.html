Why does each instruction line look like that?
  • PC: 0x80000000 — the program counter, where the fetch happens. Starts at RAM_BASE.
  • Instruction word — the raw 32 bits in hex.
  • Mnemonic — what decoder.decode turned that word into.
  • Effect — what execute.dispatch does in the matching arm.
  • The cyan-highlighted line is what will execute next.
