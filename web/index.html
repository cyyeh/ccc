      This page fetched ccc.wasm (a few hundred KB of WebAssembly
      with hello.elf embedded at compile time), instantiated it
      with no imports, and called the wasm run() export. UART
      output was captured into a buffer inside wasm linear memory; this page
      copied it out via outputPtr() and outputLen().
      No server, no remote execution, no JavaScript dependencies — your
      browser is the RISC-V machine.
